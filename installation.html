Установка Self-Hosted аналитики

Self-hosted системы аналитики позволяют владельцу сайта полностью контролировать сбор, хранение и обработку данных посетителей без передачи информации сторонним сервисам.

Пример установки Matomo
  1. Скачайте последнюю версию Matomo с официального сайта.
  2. Разархивируйте пакет на сервере в папку веб-приложения.
  3. Создайте базу данных MySQL для Matomo.
  4. Запустите веб-установщик и следуйте инструкциям.
  5. После завершения установки получите код для вставки на ваш сайт.

Процесс установки других self-hosted платформ (Plausible, Umami, Open Web Analytics) аналогичен и подробно описан в их документации.
